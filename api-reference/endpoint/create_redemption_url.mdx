<ParamField body="gift_id" type="integer" required="true">
Variable containing Thnks id of the item(s) the Customer wants to send

The Account Manager will be responsible for providing the Customer with the ID(s) for a specific Thnks or set of Thnks needed for the Webhook set up, but the Customer can also find this themselves using the below instructions:
1. Go to app.thnks.com and log in.
2. Find the Thnks you wish to send and click on the title or image
3. Once on the Thnks Details page, look for the Thnks ID here

    />
4. The number shown in the picture is your Thnks ID, which is “42997” above for a 'A Caffeine Boost'
