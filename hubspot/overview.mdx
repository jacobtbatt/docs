## How It Works

### Automatic Synchronization

Once connected, the integration automatically synchronizes your Thnks activity with HubSpot:

1. **Initial Backfill**: All historical Thnks transactions are imported into HubSpot as custom object records
2. **Hourly Sync**: New transactions are synchronized every hour automatically
3. **Status Updates**: Thnks status changes are tracked and updated in real-time
4. **Auto-Association**: Thnks records are automatically linked to matching Contacts and Deals based on email addresses

### What Gets Synced

Each Thnks transaction synced to HubSpot includes:

| Field | Description |
|-------|-------------|
| Gift Name | The name of the Thnks Thnks sent |
| Gift Amount | Dollar value of the Thnks |
| Sender | Name and email of the sender |
| Recipient | Name and email of the recipient |
| Status | Current status (Sent, Delivered, Opened, Redeemed, etc.) |
| Message | Personal message included with the Thnks |
| Sent Date | When the Thnks was sent |
| Status Dates | Timestamps for each status change |
| Tracking URL | Direct link to the Thnks transaction |

## Prerequisites

Before setting up the integration, ensure you have:

- **Thnks Account**: Enterprise or Enterprise+ plan with HubSpot integration enabled
- **HubSpot Account**: Professional or Enterprise tier (required for custom objects)
- **Admin Access**: HubSpot Super Admin permissions for initial setup
- **Matching Emails**: User email addresses should match between Thnks and HubSpot
